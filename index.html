Setup Instructions

1. Get Spreadsheet ID

Open your Google Sheet → Copy ID from URL:
docs.google.com/spreadsheets/d/[THIS_IS_THE_ID]/edit

2. Get API Key

1. Go to Google Cloud Console
2. Create a new project (or select existing)
3. Enable "Google Sheets API"
4. Go to "Credentials" → "Create Credentials" → "API Key"
5. Copy the API key

3. Share Your Sheet

In Google Sheets: Click "Share" → Make it accessible to "Anyone with the link" (Viewer) OR share with the service account email if using service account authentication.

4. Configure Here

• Spreadsheet ID: Paste the ID from step 1
• Sheet Name: Usually "Sheet1" (check your tab name)
• API Key: Paste the key from step 2
