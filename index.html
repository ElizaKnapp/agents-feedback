Setup Instructions

1. Get Spreadsheet ID

Open your Google Sheet → Copy ID from URL:
docs.google.com/spreadsheets/d/[THIS_IS_THE_ID]/edit

2. Choose Push Method

Option A: Google Apps Script (No OAuth - Recommended!)
1. Open your Google Sheet
2. Go to Extensions → Apps Script
3. Paste this code:

function doPost(e) {
  const data = JSON.parse(e.postData.contents);
  const ss = SpreadsheetApp.openById(data.spreadsheetId);
  const sheet = ss.getSheetByName(data.sheetName) || ss.getActiveSheet();
  sheet.clear();
  if (data.values && data.values.length > 0) {
    sheet.getRange(1, 1, data.values.length, data.values[0].length).setValues(data.values);
  }
  return ContentService.createTextOutput(JSON.stringify({success: true}));
}

4. Click "Deploy" → "New deployment"
5. Type: "Web app"
6. Execute as: "Me"
7. Who has access: "Anyone"
8. Click "Deploy" → Copy the Web App URL

Option B: OAuth Token (Direct API)
1. Go to OAuth 2.0 Playground
2. Select "Google Sheets API v4" → "https://www.googleapis.com/auth/spreadsheets"
3. Authorize → Exchange for tokens → Copy access token

3. Share Your Sheet

For Apps Script: No sharing needed (runs as you)
For OAuth: Share with the Google account you authenticated with

4. Configure Here

• Spreadsheet ID: Paste from step 1
• Sheet Name: Usually "Sheet1"
• Choose method (Apps Script or OAuth)
• Enter Apps Script URL OR OAuth Token
• API Key: Optional, only for Pull from public sheets
